cara membuat modules di GOlang

github
-create repo
-salin url

file
-create new file
-open vscode

vscode
-buka terminal
-go mod init https://github.com/rrrelll/belajar-golang.git
-module terbuat di go.mod
-buat file contoh: sayhello.go dan buat mereturnkan string hello
-push ke git

github
-git init
-git status
-hubungkan ke github: git remote add origin https://github.com/rrrelll/belajar-golang.git
-git add. atau git add nama file
-git commit -m "deskripsi commit"
-git push -u origin main atau master
